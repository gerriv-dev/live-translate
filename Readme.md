![Live-Translate](header.png)

Das Projekt stellt eine App bereit, die den MediaMTX-Media-Server verwendet, um eine Audio-Live-Übertragung zu ermöglichen. Dies dient der Echtzeit-Übersetzung in der Bibelgemeinde Gifhorn.

Mithilfe von FFMPEG wird der Input vom angegebenen Mikrofon direkt über den MediaMTX-Server bereitgestellt, sodass er von der Web-App wiedergegeben werden kann.
